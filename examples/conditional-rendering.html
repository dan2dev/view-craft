How it works

Boolean functions can be placed in ANY position within the modifiers:

// Boolean as first modifier
div(() => condition, "content", { className: "test" })

// Boolean as second modifier  
div("content", () => condition, { className: "test" })

// Boolean as third modifier
div("content", { className: "test" }, () => condition)

// Boolean in middle of multiple modifiers
div("before", { id: "test" }, () => condition, span("after"), { className: "test" })

When the condition is true, the element is shown with all other modifiers applied. When false, it's replaced with a comment placeholder.

Note: If boolean functions are only mixed with primitives (strings, numbers), they're treated as reactive text content instead of conditional rendering.
